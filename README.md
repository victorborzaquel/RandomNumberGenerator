# Random Number Generator

## React Native App

Aplicativo para fazer sorteios gerando um número aleatório entre dois números.  

Utilizei o date-fns para formatar a data e o i18n-js para formatar o número.

## Tecnologias

- date-fns
- i18n-js
- React Native
- Styled Components
- Typescript
